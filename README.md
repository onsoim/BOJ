# 0x0 폴더 구조

문제 풀이를 완료한 문제의 경우, 문제 번호에 따라 `01000-10000` 와 `10001-20000` 로 분류한다.

문제 풀이를 진행 중인 경우, `unsolved` 에서 진행하며 비정기적으로 갱신한다.

# 0x2 BOJ - GIT 동기화
